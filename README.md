# Modern Calculator App

A responsive modern calculator built with plain HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: `+`, `-`, `*`, `/`, `%`
- Decimal support and input validation
- Clear and delete functions
- Keyboard support for numbers, operators, `Enter`, and `Backspace`
- Calculation preview while typing
- History panel with clickable reuse
- Dark/light mode toggle
- Sound feedback on button press

## Run locally

1. Open the `calculator` folder in VS Code.
2. Open `index.html` in the browser or use Live Server.

## Deployment

This project can be hosted as a static site on GitHub Pages, Vercel, or Netlify.
