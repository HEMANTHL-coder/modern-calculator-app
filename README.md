# 🧮 Modern Calculator Web App

A modern, responsive calculator built using **HTML, CSS, and JavaScript**, featuring a clean UI, advanced functionality, and real-world best practices.

## 🚀 Features

### ✅ Core Features

* Basic operations: Addition, Subtraction, Multiplication, Division, Percentage
* Real-time expression evaluation
* Error handling (invalid input, divide by zero)

### 💡 Advanced Features

* 🧠 Safe expression parser (**No eval used**)
* ⌨️ Full keyboard support
* 📜 History panel with reuse functionality
* 🌙 Dark mode toggle
* 🔊 Sound feedback toggle
* 📱 Fully responsive design

---

## 🎨 UI Highlights

* Glassmorphism-inspired modern design
* Smooth animations and hover effects
* Operator highlighting for better UX

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout (Flexbox + Grid)
* **JavaScript (ES6+)** – Logic & Interactivity
